# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview

The Sales Data Analysis Dashboard is a Python-based data analytics project developed using Object-Oriented Programming (OOP). It loads sales data from a CSV file, cleans the data, performs business analysis, generates visual charts, and exports reports.

This project demonstrates data preprocessing, exploratory data analysis (EDA), report generation, and visualization using Python libraries.

---

## 🚀 Features

- Load CSV sales dataset
- Clean missing values and duplicates
- Calculate total revenue
- Monthly sales analysis
- Category-wise sales analysis
- City-wise sales analysis
- Top-selling products
- Top customers
- Payment mode analysis
- Discount analysis
- Generate charts automatically
- Export CSV reports
- Generate text summary report

---

## 🛠 Technologies Used

- Python 3
- Pandas
- NumPy
- Matplotlib
- OpenPyXL
- Jupyter Notebook
